# AI-Based Resume Ranker

An AI-powered system to rank resumes based on their relevance to a job description using a large language model.

## Features
- Ranks resumes by similarity to job descriptions using **Gemma 2B LLM**.
- Accepts user input in **text or PDF format**.
- Uses **NLP preprocessing**: tokenization, stopword removal, etc.
- Provides **interactive UI** with **Gradio** for easy testing.
- Achieves **85%+ relevance accuracy** on test datasets.

## Tech Stack
- **Language:** Python  
- **Libraries:** Hugging Face Transformers, Gradio, NumPy, Pandas, Scikit-learn  
- **Frontend:** Gradio interactive interface
